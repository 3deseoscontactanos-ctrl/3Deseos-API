Política de Privacidad de 3Deseos-API

Última actualización: 20 de marzo, 2026

Esta Política de Privacidad describe nuestras políticas y procedimientos sobre la recopilación, uso y divulgación de su información cuando utiliza nuestro servicio a través de la conexión con TikTok y Make.com.

1. Definiciones

  • Compañía: Se refiere a 3Deseos-API, con sede en Colombia.
  • Servicio: Se refiere a la integración de automatización de contenido para TikTok.
  • Datos Personales: Cualquier información relacionada con un individuo identificado o identificable.

2. Recopilación de Datos

Nuestra aplicación actúa únicamente como un puente técnico entre archivos de video (alojados por el usuario en Google Drive u otros) y la plataforma de TikTok. Solo procesamos:

  • Nombre de usuario de TikTok (para identificación de cuenta).
  • Archivos de video subidos por el usuario para su publicación.
  • Tokens de acceso proporcionados por TikTok (almacenados de forma segura en Make.com).

3. Uso de los Datos

Los datos se utilizan exclusivamente para:

  • Proporcionar y mantener el servicio de publicación automática.
  • Gestionar las solicitudes de publicación del usuario.

4. Seguridad

La seguridad de sus datos es importante para nosotros. Utilizamos los estándares de seguridad de Make.com y la API oficial de TikTok para garantizar que su información esté protegida durante la transmisión.
